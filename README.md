# Premium Floating AI Chatbot Widget 🚀

A modern, responsive, premium customer support chatbot widget built with an asynchronous JavaScript frontend and a secure PHP backend running the high-performance **Gemini 3.5 Flash** engine.

Designed as a drop-in component to effortlessly deploy custom business-tuned AI support automation to any website.

---

## ✨ Features Implemented

*   **Floating Action Button (FAB) Toggle:** Persistent, minimalist floating launcher bubble with custom SVG micro-interactions.
*   **Smooth Keyframe Animations:** Seamless fade and slide-up structural UI entries.
*   **Proactive Greeting Popup (Tooltip Pro):** Time-delayed attention-grabbing notification to drive user engagement.
*   **Local Auto-Welcome Handler:** Instant welcome greetings dispatched client-side without consuming API context tokens.
*   **Real-time Timestamp Labels:** Muted relative system clock calculations appended natively under text dialogue wrappers.
*   **Pulsating Status Avatar:** Branded circular assistant profile picture with an active pulsing online status beacon.
*   **Visual Typing Indicator:** Responsive animated bouncing dot structures that keep users engaged during async cURL network turns.
*   **Custom Minimalist Scrollbars:** Webkit-tailored tracking scroll rails matching modern design patterns.
*   **Fully Mobile Responsive:** Media queries that instantly scale the chat framework to full-screen execution layouts on small mobile viewports.

---

## 🛠️ Tech Stack & Architecture

*   **Frontend:** HTML5, CSS3 (Flexbox/Keyframe Animations), Asynchronous JavaScript (Native Fetch API API interactions).
*   **Backend:** PHP (Native stream processing, cURL implementation).
*   **AI Engine:** Google AI Studio API (`gemini-3.5-flash`).
